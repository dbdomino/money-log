# skills

프로젝트 AI 스킬을 여기에 추가한다.

## 새 스킬 추가

1. `skills/<skill-name>/` 디렉터리를 만든다.
2. 아래 템플릿으로 `SKILL.md` 를 작성한다.
3. 필요하면 `reference.md`, `examples.md`, `scripts/` 를 추가한다.

```markdown
---
name: skill-name
description: 무엇을 하는지. Use when 사용자가 ~를 요청할 때.
---

# Skill Title

## Instructions
1. ...
2. ...

## Examples
- ...
```
