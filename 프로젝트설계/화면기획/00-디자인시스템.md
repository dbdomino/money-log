# [화면기획] 디자인 시스템

> 대상: `money-app` (Thymeleaf) · CSS는 [`css/tokens.css`](./css/tokens.css), [`css/ui.css`](./css/ui.css)  
> 이식: `src/main/resources/static/css/`

## 원칙

- 심플·눈이 편한 **그린** 계열. 과한 그라데이션·글로우·카드 남발 금지.
- 목록은 **테이블 + 툴바**, 등록·수정·상세는 **모달**.
- 포커스 링·대비는 primary 그린으로 통일.

## 컬러

| 토큰 | 값(대표) | 용도 |
|------|----------|------|
| `--color-bg` | `#f4f7f5` | 페이지 배경 |
| `--color-surface` | `#ffffff` | 패널·모달·테이블 |
| `--color-primary` | `#2f6f5e` | CTA·활성 메뉴 |
| `--color-primary-soft` | `#dcebe4` | 호버·정보 알림 |
| `--color-text` | `#2a322f` | 본문 |
| `--color-text-secondary` | `#5c6b64` | 라벨·보조 |
| `--color-border` | `#c5d4cc` | 구분선 |
| `--color-danger` | `#b54a3c` | 삭제·오류 |

뱃지: 지출 빨강계 · 소득 초록계 · 고정 보라계 · 할부 주황계 (`tokens.css` 참고).

## 타이포·간격

- 폰트: Pretendard / Noto Sans KR 우선, 시스템 산세리프 폴백.
- 기본 15px, 본문 line-height 1.5.
- 간격: 4 / 8 / 12 / 16 / 24 / 32 (`--space-*`).
- radius: 6 / 10 / 14.

## 컴포넌트

| 이름 | 클래스 | 비고 |
|------|--------|------|
| Primary 버튼 | `.btn.btn-primary` | 저장·로그인 |
| Secondary | `.btn.btn-secondary` | 취소·보조 |
| Ghost | `.btn.btn-ghost` | 테이블 행 액션 |
| 입력 | `.input` / `.select` / `.textarea` | `name` = API 필드명 |
| 툴바 | `.page-toolbar` 또는 `.toolbar` + `.field` | 필터·조회. select는 auto width |
| 패널 | `.panel` / `.panel--narrow` / `.panel--md` | 설정형 폼(목표·프로필·엑셀) |
| 테이블 | `.table-wrap` > `.table` | 호버 행 강조. 액션열 헤더 `관리` |
| 뱃지 | `.badge.badge-*` | 가계부 구분. 기본 `.badge`는 muted(흰글씨 금지) |
| 모달 | `.modal-root.is-open` | 딤 + 패널. wide=`is-lg`/`--wide`, 확인=`--sm` |
| 요약 | `.summary-row` / `.summary-item` | 가계부 합계 |

## 모달 UX

1. 열기: 버튼 `data-modal-open="id"` 또는 URL `?m=...`
2. 닫기: 취소·X·Esc·딤 클릭
3. 등록/수정: footer에 취소 + 저장
4. 상세: footer에 닫기 + 수정(모달 전환) + 삭제(확인 다이얼로그)

## 접근성

- 포커스 가능 컨트롤에 `:focus-visible` 링.
- 모달 오픈 시 패널로 포커스 이동(이식 시 JS).
- 아이콘만 있는 버튼은 `aria-label` 또는 `.sr-only` 텍스트.
