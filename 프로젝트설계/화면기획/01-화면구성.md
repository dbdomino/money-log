## URL 전략

- **대표 URL** = 풀페이지만.
- 구 URL (`/payments/new`, `/ledger/expenses/new` 등) → 부모 + `?m=` 로 **redirect** (구현 시).
- 모달 없이 직접 열리면 안 되는 작업은 부모 목록으로 보낸 뒤 모달 오픈.

## 본문 패턴

```
page-header (제목 + 등록 버튼)
page-toolbar (필터·검색)
[summary-row]   ← 가계부·통계
table-wrap
pagination
modal-root × N
```
