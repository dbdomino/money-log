## 응답

공통 래퍼: [_공통.md § 응답 래퍼](../_공통.md#응답-래퍼)  
목록 표준: [_공통.md § 목록 응답 규칙](../_공통.md#목록-응답-규칙) (`data.list` = object 배열)  
공유 타입: [_공통.md § PaymentMethod](../_공통.md#공유-타입--paymentmethod)

### 성공 (`resCode: 200`) — `data`

| 필드 | 타입 | 설명 |
|------|------|------|
| `list` | array (object[]) | 본인 수단 전체 목록 |
| `list[].paymentMethodId` | number | PK |
| `list[].name` | string | 이름 |
| `list[].type` | string | `CARD` \| `ACCOUNT` |
| `list[].inUse` | boolean | 사용 여부 |
| `list[].cardExpiry` | string \| null | |
| `list[].deleted` | boolean | soft delete 여부 |

### 실패 — 대표 `resCode`

| resCode | 조건 |
|---------|------|
| 1001 | 로그인 필요 |

## 비고

- 목록 응답 표준: `data.list` = **object 배열** ([_공통 § 목록 응답 규칙](../_공통.md#목록-응답-규칙)).
- **전체 목록**(설정·관리 화면). 삭제(`deleted=true`) 건도 포함.
- 페이징 없음 (본인 수단 수 제한적).
- 사용 중인 수단만 필요하면 `PaymentMethodListActive` 사용.
