# [백엔드] 기본 목표금액 등록·수정

> API: `ExpendTargetDefaultUpsert` · Phase 5 · 기능번호 5.3  
> 인덱스: [README.md](../README.md)

## 메타

| 항목 | 내용 |
|------|------|
| Phase | 5 (목표금액·통계) |
| 기능 이름 | 기본 목표금액 등록·수정 |
| 기능번호 | 5.3 |
| API 이름 | `ExpendTargetDefaultUpsert` |
| Method | `PATCH` |
| URL | `/api/v1/expend-targets/default/{expendGroupId}` |
| 권한 | 로그인 |
| Content-Type | `application/json` |

## 요청

### Path Variables

| 이름 | 타입 | 필수 | 설명 |
|------|------|:----:|------|
| `expendGroupId` | number | ✅ | 지출유형 PK |

### Body

| 필드 | 타입 | 필수 | 설명 |
|------|------|:----:|------|
| `defaultTargetAmount` | number | ✅ | **월 단위** 기본(템플릿) 목표금액 (한 달에 쓸 한도). 0~100,000,000 |

### Headers

| 이름 | 필수 | 설명 |
|------|:----:|------|
| `Authorization` | ✅ | `Bearer <accessToken>` |
| `Content-Type` | ✅ | `application/json` |

## 응답

### 성공 (`resCode: 200`) — `data`

| 필드 | 타입 | 설명 |
|------|------|------|
| `expendGroupId` | number | 지출유형 PK |
| `expendGroupName` | string | 지출유형 이름 |
| `defaultTargetAmount` | number | 저장된 **월 단위** 기본 목표금액 |

### 실패 — 대표 `resCode`

| resCode | 조건 |
|---------|------|
| 1001 | 로그인 필요 |
| 3103 | 유형 없음 |
| 3601 | **미사용(inUse=false) 유형** |
| 3602 | 목표금액 범위 초과 |

```http
  }
```


## 비고

- 최초 설정·변경 모두 PATCH upsert. **기본만** 변경하며, 이미 저장된 월별 목표는 바꾸지 않는다.
- `defaultTargetAmount`는 **월 단위** 금액이다. 일·주 목표가 아니라, 해당 지출유형에 대해 **한 달 기준**으로 쓸 목표 한도이다.
- 특정 연·월에 월별 목표가 없으면 통계 등에서 이 기본값을 **그달 목표**로 사용한다.
- 기본·해당월 금액을 함께 보려면 [`ExpendTargetGet`](./5.2-ExpendTargetGet.md) 또는 [`ExpendTargetList`](./5.1-ExpendTargetList.md)로 재조회한다.
