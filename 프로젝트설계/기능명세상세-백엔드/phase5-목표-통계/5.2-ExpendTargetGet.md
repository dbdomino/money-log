# [백엔드] 목표금액 상세 조회

> 인덱스: [README.md](../README.md)

| 항목 | 내용 |
|------|------|
| Phase | 5 (목표금액·통계) |

## 요청

### Path Variables

| 이름 | 타입 | 필수 | 설명 |
|------|------|:----:|------|
| `year` | number | ✅ | 조회 연도. 월별 목표(`monthlyTargetAmount`) 기준 연도 |
| `month` | number | ✅ | 조회 월 (1~12). 월별 목표 기준 월 |
| `expendGroupId` | number | ✅ | 지출유형 PK |

### Query Parameters

없음

### Headers

| 이름 | 필수 | 설명 |
|------|:----:|------|
| `Authorization` | ✅ | `Bearer <accessToken>` |

### Body

없음

## 응답

공통 래퍼: [_공통.md § 응답 래퍼](../_공통.md#응답-래퍼)

### 성공 (`resCode: 200`) — `data`

| 필드 | 타입 | 설명 |
|------|------|------|
| `year` | number | 조회 연도. Path `year`와 동일 |
| `month` | number | 조회 월. Path `month`와 동일 |
| `expendGroupId` | number | 지출유형 PK |
| `expendGroupName` | string | 지출유형 이름 |
| `defaultTargetAmount` | number | **기본(템플릿)** 월 목표금액 (0~100,000,000). 미설정 시 `0` |
| `monthlyTargetAmount` | number \| null | Path `year`/`month`에 **별도 저장된** 월별 목표. 없으면 `null` (기본값을 복사하지 않음) |

### 실패 — 대표 `resCode`

| resCode | 조건 |
|---------|------|
| 1001 | 로그인 필요 |
| 3103 | 유형 없음 또는 타인 소유 |
| 3601 | **미사용(inUse=false) 유형** (목표 화면에서 상세 불가) |
| 3603 | 월 범위 오류 (1~12) 등 연·월 검증 실패 |

## 예시

### 성공

**Request**

```http
GET /api/v1/expend-targets/2026/7/1 HTTP/1.1
Host: localhost:8081
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

**Response**

```json
{
  "resCode": 200,
  "data": {
    "year": 2026,
    "month": 7,
    "expendGroupId": 1,
    "expendGroupName": "식비",
    "defaultTargetAmount": 400000,
    "monthlyTargetAmount": 500000
  }
}
```

### 실패

```json
{
  "resCode": 3601,
  "data": {
    "message": "사용 중이지 않은 지출 유형에는 목표금액을 조회할 수 없습니다"
  }
}
```

## 비고

- 목록([`ExpendTargetList`](./5.1-ExpendTargetList.md))과 **동일 금액 필드** (`defaultTargetAmount`, `monthlyTargetAmount`).
- Path만 사용 (Query 혼용 없음). 월별 upsert는 `/monthly/...` 하위 경로라 충돌하지 않는다.
- 수정 후 전체 확인은 본 API 또는 목록 재조회. upsert 응답은 담당 필드만 반환.
- 기본 수정: [`ExpendTargetDefaultUpsert`](./5.3-ExpendTargetDefaultUpsert.md). 월별 수정: [`ExpendTargetMonthlyUpsert`](./5.4-ExpendTargetMonthlyUpsert.md).
- 적용 금액(통계 등): `monthlyTargetAmount ?? defaultTargetAmount`.
